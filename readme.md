# 🎓 Student Mood Chatbot

## Overview

Student Mood Chatbot is a simple AI-based chatbot that interacts with students and provides helpful suggestions based on their mood.

## Features

- Detects student mood
- Provides study tips
- Gives motivational suggestions
- User-friendly interaction

## Technologies Used

- Python
- Rule-Based AI
- Conditional Statements

## How to Run

```bash
python chatbot.py
```
